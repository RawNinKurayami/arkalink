# Grand Line Chronicles
Webapp del gioco di ruolo "Grand Line Chronicles" (sistema Will of D., fan game di One Piece).

## Struttura
- `index.html` — home (carta "Il mio Profilo" + accesso)
- `glc-auth.js` — accesso condiviso (Google + link magico, salvataggi cloud)
- `profilo/` — profilo social: avatar, nome, lista dei propri pirati
- `crea-il-tuo-pirata/` — creazione guidata di un nuovo pirata
- `gestisci-pirata/` — gestione & avanzamento (?c=ID)
- `nave-portale/`, `portale-officina-cambusa/`, `plancia-master/` — altri strumenti
