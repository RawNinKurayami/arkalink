# Grand Line Chronicles

Webapp del gioco di ruolo "Grand Line Chronicles" (sistema **Will of D.**, fan game di One Piece).

## Struttura
- `index.html` — la home / porto della ciurma
- `crea-il-tuo-pirata/` — Scheda del Pirata (creazione e gestione personaggio)
- `nave-portale/` — gestione della Nave
- `portale-officina-cambusa/` — Officina & Cambusa (crafting, veleni, cucina)
- `plancia-master/` — Plancia del Master (bestiario + tracker scontri)

Pagine statiche, pubblicate con GitHub Pages.
