# Grand Line Chronicles
Webapp del gioco di ruolo "Grand Line Chronicles" (sistema Will of D., fan game di One Piece).

## Struttura
- `index.html` — home / porto della ciurma (con accesso)
- `glc-auth.js` — modulo di accesso condiviso (Google + link magico, salvataggi nel cloud)
- `profilo/` — profilo: nome utente + lista dei propri pirati (Apri scheda / Nuovo)
- `crea-il-tuo-pirata/` — Creazione guidata di un nuovo pirata, passo per passo
- `gestisci-pirata/` — Gestione & avanzamento di un pirata della lista (?c=ID)
- `nave-portale/`, `portale-officina-cambusa/`, `plancia-master/` — gli altri strumenti

Pagine statiche su GitHub Pages. Login e salvataggi via Supabase.
